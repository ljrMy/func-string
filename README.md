# About

## Operator
1. `+` -> Plus | 加
2. `-` -> Minus  | 减
3. `*` -> Times | 乘
4. `/` -> Divided by | 除
5. `%` -> Modulus | 模
6. `^` -> Power | 幂
7. `\` -> Root | 开方
8. `_` -> Logarithm | 对数
9. `:` -> Round | 四舍五入
10. `)` -> Call a built-in function | 调用内置函数

## Example
If a func-string is `-1%30+1`,
1. Subtract 1. | 减去1
2. Modulus by 30. | 模除30
3. Add 1. | 加上1

If a func-string is `\2:3`,
1. Take the square root of. | 开平方根
2. Round to three decimal places. | 四舍五入保留小数点后3位。
